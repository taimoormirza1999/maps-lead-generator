# Project Rules

## Permissions
- Auto-approve Grep, Read, Write tools and Bash commands (grep, cat, head, tail, awk, sed) without asking for permission.

## Style
- No robotic menus or bot-like messages. Use short, natural, casual tone.
- Use claude -p CLI for AI responses, not separate API.

## Shared Services
- WhatsApp sessions: see skill `wa-service`
- Email sending: see skill `email-service`
- LinkedIn session: see skill `linkedin-service`
- Instagram / Facebook / TikTok: see skill `social-service`
- Daily outreach pipeline: see skill `daily-outreach`
